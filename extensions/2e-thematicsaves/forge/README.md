# 2E Thematic Saves

FGU extension for the official 2E (AD&D 2nd Edition) ruleset.

Relabels the ten expanded saving throws with shorter, thematic names and adds tooltips. Mechanics, scores, and database keys are completely unchanged — this only affects labels and chat text.

## Name Map

| Stock Name | Thematic Name |
|---|---|
| Paralyzation | Hold |
| Poison | Poison |
| Death | Death |
| Rod | Fear |
| Staff | Device |
| Wand | Ray |
| Petrification | Stone |
| Polymorph | Curse |
| Breath | Blast |
| Spell | Spell |

## How to Use

Just enable the extension and load a 2E campaign — character sheet labels, class advancement headers, and chat roll messages automatically show the thematic names. No other setup or interaction is required.

## Compatibility

- Official 2E (AD&D 2nd Edition) ruleset
- Safe alongside 2E Attack Throw, 2E Ascending AC, 2E Target 20, and similar extensions — labels and strings only, no mechanics touched
