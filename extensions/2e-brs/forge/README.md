# 2E BRS - Better Races & Subraces

FGU extension for the official 2E (AD&D 2nd Edition) ruleset.

Adds Proficiencies, Non-Weapon Proficiencies (Skills), and Advanced Effects sections to Subrace records — previously limited to Traits only — and applies them to a character automatically when that subrace is chosen. Also adds a Non-Weapon Proficiencies section to the base Race record.

## How to Use

1. Open any Race or Subrace record with the extension loaded.
2. Race records now show a **Non-Weapon Proficiencies** section below the existing weapon Proficiencies section.
3. Subrace records now show all three: **Proficiencies**, **Non-Weapon Proficiencies**, and **Advanced Effects**, below Traits.
4. Drag a Skill reference record into Non-Weapon Proficiencies, a proficiency reference into Proficiencies, or author Advanced Effects entries directly — the same way you already would on a Race record.
5. When a character is given that Race and a subrace is chosen (or auto-selected, if the race has only one), everything authored on both the base race and the chosen subrace is applied together automatically.

## Compatibility

- Official 2E (AD&D 2nd Edition) ruleset
- Purely additive — doesn't remove or replace any stock Race/Subrace functionality
